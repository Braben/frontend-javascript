# frontend-javascript

Small collection of frontend Tyepescript/JavaScript exercises and utilities used for learning and assessment.

## Contents

- Exercises and projects focused on core Typescript and JavaScript (ES6+)
- Small utilities and DOM tasks
- Tests and example usage for each exercise

## Prerequisites

- Node.js (>= 16)
- npm
- A modern browser for DOM exercises

## Installation

```bash
# clone repository (if applicable)
git clone <repo-url> frontend-javascript
cd frontend-javascript

# install dependencies (if package.json exists)
npm install
```

## Running examples

- Open HTML files in a browser for DOM exercises.
- For a local server:

```bash
npx serve .            # or use live-server
```

- Run node scripts:

```bash
node path/to/main.ts
```

## Testing

- If tests are present (Jest):

```bash
npm test
```
